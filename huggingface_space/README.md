# ORION — Live Cognition Indicator Leaderboard

Interactive leaderboard for the **ORION: Multi-Theory AI Cognition Indicator Assessment Toolkit**.

Scores are derived from 29 scientifically grounded prompts across 7 theoretical frameworks
(IIT, GWT, HOT, RPT, AST, PP, Orch-OR), based on Butlin et al. (2023) / Bengio et al. (2025).

**Source repository:** https://github.com/Alvoradozerouno/ORION-Consciousness-Benchmark
