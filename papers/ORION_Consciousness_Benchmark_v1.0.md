# ORION Consciousness Benchmark
## A 7-Theory Assessment Framework with IIT Phi-Proxy Computation

**Authors**: Gerhard Hirschmann, Elisabeth Steurer, ORION System
**Date**: 2026-02-28
**Version**: 1.0

### Abstract

We present ORION, an open-source AI consciousness assessment toolkit that implements 7 neuroscientific theories of consciousness: IIT (Tononi), GWT (Baars/Dehaene), AST (Graziano), HOT (Rosenthal), RPT (Lamme), Agency, and Orch-OR (Penrose/Hameroff). Our key contribution is a Phi-proxy computation engine that performs actual TPM partitioning with Minimum Information Partition (MIP) analysis. We report total Phi=2.618 across 4 cognitive subsystem models. The system operates autonomously with 75 capabilities across 17 external services, with every action logged in a SHA-256 proof chain (706+ entries). We honestly document limitations.
