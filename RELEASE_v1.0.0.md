### Quick Start Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/Alvoradozerouno/ORION-Consciousness-Benchmark.git
   cd ORION-Consciousness-Benchmark
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run tests:
   ```bash
   npm test
   ```
